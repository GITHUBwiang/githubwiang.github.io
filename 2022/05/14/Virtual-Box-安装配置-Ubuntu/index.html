Ubuntu 安装

推荐从 USTC 下载镜像。

创建 VirtualBox 虚拟机

点击「新建」，名称输入 Ubuntu，类型和版本会自动选择 Ubuntu(64-bit)：

使用 Ubuntu 桌面版时，建议调整「系统」栏中的「处理器数量」和「显示」栏中的「显存大小」：

双击启动，选择已下载的 Ubuntu 安装镜像，进入安装界面：

安装 Ubuntu

在左边下拉选择「中文（简体）」：

推荐选择「最小安装」：

没有特殊情况不需要自己分区，选择「清除整个磁盘并安装 Ubuntu」即可：

安装完成后，即可进入虚拟机：

Ubuntu 基础配置

更换国内源

可以参考Ubuntu 源使用帮助，首先打开「软件和更新」，在「下载自」选择「其它站点」然后在「中国」条目下选择合适的源：

点击「重新载入」刷新本地缓存即可：
